# Niro – Farm Bot

Automatyczny bot do gry WolniFarmerzy + panel webowy (FastAPI + JS)

## Tech stack
- Python (FastAPI)
- JavaScript (Vanilla)
- HTML/CSS

## Funkcje
- automatyczne farmienie
- sadzenie i zbiór
- obsługa ogrodu

## Jak uruchomić

### Backend
uvicorn backend.main:app --reload

### Frontend
cd frontend
python -m http.server 5500
